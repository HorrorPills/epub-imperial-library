# The Elder Scrolls Tomes

A collection of in-game books from *The Elder Scrolls* series, compiled from [The Imperial Library](https://www.imperial-library.info/game-books) and packaged as four Kindle-ready EPUB files.

> **Journals, Notes, and Letters are excluded from all volumes.**
> Each tome contains every other book available in that game, including expansions and DLC.

## Source & Script

Books sourced from **[The Imperial Library](https://www.imperial-library.info)** — the definitive Elder Scrolls lore archive.

The `scrape_elder_scrolls.py` script automates the full pipeline:
- Scrapes all four game listing pages
- Skips Journals, Notes, and Letters sections
- Deduplicates books by URL
- Fetches content from each individual book page
- Packages everything into Kindle-ready EPUBs with a styled table of contents

```bash
python3 scrape_elder_scrolls.py
```

All required packages (`requests`, `beautifulsoup4`, `lxml`, `ebooklib`) are installed automatically on first run.
